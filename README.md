# DND-Manager

##Overview

This project simulates a simplified **DND Manager** used for the Dungeons & Dragons game.

The system takes input from the Client and logs it in the database for client and server side both, which includes the following:

For Players:
- Current Player Selection
- Dice Roll

For Dungeon Masters:
- Applying effects on the Players or the Monsters
- Changing the HP (Health Point) Status in +/-

The goal is to build a **clean, client and server based system** which provides a functionable DND Character Sheet which keeps the track of everything which is logged in it.
